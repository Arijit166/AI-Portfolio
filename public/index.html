Featured Projects

InvestorHunt

InvestorHunt is a platform connecting entrepreneurs and investors, featuring role-based authentication, project showcasing, and investor dashboards. Built with Next.js and MongoDB, it simplifies startup funding through intuitive matchmaking and secure communication tools.

Trinoyon

Trinoyon is an interactive fundraising website that combines cultural engagement with social impact. It features games, quizzes, a Pujo gallery, and a virtual Pujo space where users can share festival photos and contribute donations to support underprivileged communities.

SwiftCart

SwiftCart is a modern e-commerce platform built with Node.js, Express, and MongoDB. It offers a seamless shopping experience with secure authentication, admin and user dashboards, product management, and a clean, responsive UI powered by Tailwind CSS.

Linksy

Linksy is a social media platform where users can create profiles, share posts, follow others, and engage with content through likes and comments. Built with a responsive interface and scalable backend, it delivers a smooth and community-driven experience.

ClimaBot

ClimaBot is an interactive weather web app that provides real-time weather updates, sunrise/sunset times, and moon phases for any location. It features an AI chatbot that suggests activities and destinations based on current weather conditions, delivering an engaging user experience.

Space Defenders

Space Defenders is a fast-paced arcade-style shooter game where players control a spaceship to defend against waves of enemy invaders. It features smooth animations, collision detection, increasing difficulty, and responsive controls.
